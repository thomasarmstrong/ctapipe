Camera¶

class ctapipe.instrument.CameraDescription.Camera(pix_id, pix_X, pix_Y, pix_area, pix_type, pix_neighbors)[source]¶

Bases: object

Camera is a class that provides and gets all the information about the camera of a specific telescope.

Parameters:

self: type

description

pix_id: array (int)

pixel ids of the camera of the telescope

pix_X: array with units

position of each pixel (x-coordinate)

pix_Y: array with units

position of each pixel (y-coordinate)

pix_area: array with units

area of each pixel

pix_type: string

name of the pixel type (e.g. hexagonal)

pix_neighbors: ndarray (int)

nD-array with pixel IDs of neighboring pixels of the pixels (n=number of pixels)
