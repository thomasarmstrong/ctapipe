get_site_id_for_time¶

ctapipe.instrument.obsconfig.get_site_id_for_time(obstime)[source]¶

lookup which array and version was used for a given obstime (astropy.time.Time)

Example:

aid, aver = get_site_id_for_time( Time("2015-01-01", scale='utc') )
